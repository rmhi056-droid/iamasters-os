# Variantes de orden de sombreros

El orden importa. Empezar por el sombrero equivocado puede hundir la sesión.

## 1. Análisis general (por defecto)

**Orden:** Azul → Blanco → Amarillo → Negro → Verde → Rojo → Azul
**Lógica:** hechos primero, luego lo bueno, luego lo malo, luego alternativas, luego intuición. Emoción al final para no sesgar el análisis racional.

## 2. Funcionalidad de producto / software

**Orden:** Azul → Blanco → Amarillo → Verde → Negro → Rojo → Azul
**Lógica:** generar alternativas (Verde) antes de criticar (Negro), porque las críticas se aplican mejor a un conjunto de opciones.

## 3. Conflicto entre personas

**Orden:** Azul → Rojo → Blanco → Amarillo → Verde → Negro → Azul
**Lógica:** las emociones van PRIMERO porque hasta que no salen no se puede pensar. Después hechos, terreno común, opciones nuevas, y solo entonces evaluar riesgos.

## 4. Pre-mortem de proyecto

**Orden:** Azul → Blanco → Negro → Rojo → Verde → Amarillo → Azul
**Lógica:** Negro temprano porque el objetivo es encontrar fallos. Rojo después captura alarmas internas. Verde busca cómo evitar los fallos. Amarillo al final como contrapeso.

## 5. Desbloqueo creativo

**Orden:** Azul → Verde → Verde → Amarillo → Blanco → Negro → Rojo → Azul
**Lógica:** dos pasadas de Verde (la 2ª con provocación obligatoria) antes de cualquier otro modo. Negro al final para no matar opciones prematuramente. Regla extra: el Negro no puede descartar más del 50% de las ideas.

## 6. Decisión de alto coste / irreversible

**Orden:** Azul → Blanco → Amarillo → Negro → Verde → Negro → Rojo → Azul
**Lógica:** doble pasada por Negro. La primera evalúa la propuesta original; la segunda evalúa las alternativas del Verde. Previene que una alternativa nueva escape al escrutinio.

## 7. Análisis rápido (condensado)

**Orden:** Azul → Blanco → Amarillo+Negro (juntos pero separados visualmente) → Rojo → Azul
**Lógica:** se sacrifica el Verde. Única excepción al aislamiento estricto. Usar solo si el usuario lo pide explícitamente. Avisar: "Versión condensada, sin Verde ni anti-ancla completa."

## Cómo elegir

1. ¿Hay personas en conflicto? → **3**
2. ¿Es una feature de producto? → **2**
3. ¿Compromiso grande e irreversible? → **6**
4. ¿Quieres ver cómo podría fallar? → **4**
5. ¿Estás atascado? → **5**
6. ¿Tienes prisa? → **7** (con aviso)
7. Cualquier otro caso → **1**
